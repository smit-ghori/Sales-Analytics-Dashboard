# Sales Analytics Dashboard

A professional Power BI dashboard project designed to analyze sales performance, customer activity, regional distribution, and profitability from structured order data. This repository showcases a clean business intelligence workflow, from raw CSV data to an interactive dashboard experience.

## Overview

This project presents an end-to-end analytics solution using Microsoft Power BI. It focuses on transforming transactional data into meaningful business insights that can support decision-making for sales, marketing, and operations teams.

## Dataset Description

The project uses two main data files:

- details.csv: contains order-level information such as order date, customer name, state, and city
- orders.csv: contains financial and product-related details such as amount, profit, quantity, category, sub-category, and payment mode

## Dashboard Goals

The dashboard is built to help users answer questions such as:

- Which categories and sub-categories generate the highest sales?
- Which regions and cities contribute most to revenue?
- How do payment methods affect purchasing behavior?
- Which orders are profitable and which are underperforming?
- How does customer activity vary across locations and time?

## Key Features

## How to Use

1. Download or clone this repository.
2. Open the Power BI file [visulization.pbix](visulization.pbix).
3. Ensure the CSV files remain in the same project folder or update the data source paths if needed.
4. Refresh the report to load the latest data and explore the visuals.
